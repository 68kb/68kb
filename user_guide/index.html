Welcome to 68KB!

68KB is an easy to use knowledge base system. The primary goal is to keep things simple and do just enough to get the job done. Let's face it not everyone wants every single bell and whistle in a script and for that reason we kept the core to a bare minimum. That is not to say 68KB can't be easily extended. We include an advanced module system and an api. For those that want to add more features or have it perform different.

Who is 68KB For?

68KB is right for you if:

  • You want a simple easy to use knowledge base.
  • You want a knowledge base that requires nearly zero configuration.
  • You want a knowledge base that is easy to skin.
  • You need clear, thorough documentation.
