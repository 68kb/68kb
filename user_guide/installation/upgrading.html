Upgrading Instructions

These instructions will guide you go through the steps to install 68KB.

Step 1. Download and extract the files

Visit the download page on 68KB and download the latest version. After downloading it you should extract all the files to your hard drive.

Step 2. Make a Backup

Make a backup of all your files and your database. You can backup your database through the administration -> settings -> utilities. After you have made the back up please test it to make sure it will work if something should go wrong.

Step 3. Upload the files

You should now FTP into your site and upload all the contents of the 'upload' folder. After they finish uploading CHMOD the 'uploads' directory to 777 or read, write, execute all. (notice it is plural)

Step 4. Run the upgrade

The final step is to actually run the upgrade which will create the necessary tables. You will do this by visiting http://yoursite.com/index.php/setup/ and following the upgrade steps.

After you run the install PLEASE DELETE THE sources/controllers/setup DIRECTORY.
