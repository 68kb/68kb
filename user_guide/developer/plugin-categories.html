Categories Plugin

The categories plugin is located in includes/application/plugins/categories_pi.php this plugin allows you to generate a list of categories.

Usage

To use this plugin you can do so by using this code in any of your template files:

<?php categories_plugin('parent=1&depth=2'); ?>

If you wanted to show all parent categories you could use this:

<?php categories_plugin('depth=1'); ?>

Parameters

Preference Default Value Options Description
parent None Integer If you want to show any categories that are children of a certain parent.
depth 2 Integer The depth of categories you want returned. For example if you enter a depth of 1 then only parent categories will be shown. PLEASE NOTE: If you set the depth to a high number please considering using caching for better performance.
type list String Either list or table. If you choose table please see the additional params below.
show_count FALSE Boolean Do you want to display the total articles in that category. TRUE or FALSE
show_description FALSE Boolean Do you want to display the category description. TRUE or FALSE

Table Parameters

Preference Default Value Options Description
table_attr None String Any table attributes you would like. For example: table_attr=width="100%"
row_start None String Any tr attributes you would like.
cell_start None String Any td attributes you would like.
row_alt_start None String Any tr attributes you would like. This is used to alternate between rows.
cell_alt_start None String Any td attributes you would like. This is used to alternate between cells in rows.
trail_pad &nbsp; String What you want inserted in blank cells.
cols 2 Integer This is the number of columns you want to display before a new row is started.
