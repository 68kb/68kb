Articles Plugin

The articles plugin is located in includes/application/plugins/articles_pi.php this plugin allows you to generate a list of articles based off the parameters you choose.

Usage

To use this plugin you can do so by using this code in any of your template files:

<?php articles_plugin('sort=article_hits&order_by=desc&limit=10'); ?>

That code would limit to 10 articles by the article_hits column descending.

If you wanted to show the newest articles you could use this:

<?php listings_plugin('featured=yes&order_by=random'); ?>

As the final example lets say you want to display 5 random listings from three categories in 2 columns using style 2:

<?php articles_plugin('sort=article_date&limit=10'); ?>

Notice we left out the order_by=desc that is because order by desc is the default action for order_by and doesn't have to be explicitly stated. You can see all the parameters below.

Parameters

Preference Default Value Options Description
limit 10 Integer The max number to return.
author None Integer If you want to limit to a specific author.
category None Integer Which category or categories it should look in. For one category use the category id. For multiple use commas to seperate. category=1,2,3,4
keywords None String Do you want to only show listings based off a keyword.
sort article_rating column name Do you want to sort off a column name.
order_by random random, asc, desc How do you want to order the results by.
return FALSE Boolean Should the results be returned or echoed?
