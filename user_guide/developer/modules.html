Modules & Plugins

Modules and Plugins allow to create or include extra functionality that does not touch the core files. Thus enabling these files to not get overwritten during upgrades.

Modules and plugins are very similar. The main difference is that a plugin usually provides a single function, whereas a Module is usually a collection of files that extend the system.
