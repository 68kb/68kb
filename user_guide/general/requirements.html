System Requirements

For running 68KB you need to make sure your host meets the following requirements.

Server Requirements

  • PHP version 5.1 or newer.
  • MySQL (4.1+)
  • cURL or fsockopen

Other items you will need:

  • Text Editor - Some of the more popular programs are:
    • Dreamweaver - (http://adobe.com)
    • Coda - (http://www.panic.com/coda/)
    • Notepad++ - (http://notepad-plus.sourceforge.net)
    • TextMate - (http://macromates.com/)
    • BBEdit - (http://www.barebones.com/)
    • Notepad included in all versions of Windows
  • FTP Client - There are many programs currently available including:
    • Filezilla - (http://filezilla-project.org/)
    • CuteFTP - (http://www.globalscape.com/cuteftp/)
    • Transmit - (http://www.panic.com/transmit/)
    • SmartFTP - (http://www.smartftp.com/)
  • A program capable of unzipping compressed files. There are many programs currently available including:
    • WinRAR - (http://www.rarlab.com/)
    • WinZip - (http://www.winzip.com/)
