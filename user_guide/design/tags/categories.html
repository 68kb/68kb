Category Tags

68KB has two methods for getting a list of categories. They are:

  1. Categories List
  2. Categories Table

Category List

The {kb:categories:cat_list} tag is used to generate a html list of categories any where on your site.

Here is an example usage:

{kb:categories:cat_list }

Inside the block you can access any column from the db. {kb:article_hits}, {kb:article_date}, etc.

Category List Parameters

Name Type Default Description
cat_parent int 0 Is the starting parent category.
sort_column string cat_order The db column to sort
sort_order string desc The sorting order. (asc, or desc)
depth int 0 The depth of the tree you want returned.
show_total (yes or no) no Do you want to display the total number of articles. Please note this can be slow.
exclude int 0 Any categories to exclude. Example: exclude=1,2,3

Category Table

The {kb:categories:table} tag is used to generate a html table of categories any where on your site.

Here is an example usage:

{kb:categories:table heading="Categories" cat_parent="0" cols="3" show_total="yes"}

Category Table Parameters

Name Type Default Description
heading string - This creates an h2 heading above the category.
cat_parent int 0 Is the starting parent category.
sort_column string cat_order The db column to sort
sort_order string desc The sorting order. (asc, or desc)
depth int 0 The depth of the tree you want returned.
show_total (yes or no) no Do you want to display the total number of articles. Please note this can be slow.
cols int 2 Number of table columns
table_attr string width="100%" class="cat_table" Table attributes
