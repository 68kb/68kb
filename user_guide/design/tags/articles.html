Article Tags

The {kb:articles:get} tag is used to allow you to generate a list of articles any where on your site.

Here is an example usage:

<h3>Recent Articles</h3>
<ol>
{kb:articles:get sort_column="article_date" sort_order="desc"}
   <li><a href="{kb:article_url}">{kb:article_title}</a></li>
{/kb:articles:get}
</ol>

Inside the block you can access any column from the db. {kb:article_hits}, {kb:article_date}, etc.

Parameters

Name Type Default Description
limit int 10 The max number of listings.
author int - Only show articles with this author.
category int - Only show articles in this category.
sort_column string listing_title The db column to sort
sort_order string random The sorting order. (random, asc, or desc)
exclude string - Allows you to add articles to be excluded. exclude=1 or exclude=1,2,3
