Utilities

The utilities section allows you to perform various tasks related to your knowledge base.

Optimize Database

By clicking the optimize database link the script will go through your database and optimize all your tables.

Back to Top

Repair Database

In situations where your database is having issues you can click the repair database and it will repair all your tables.

Back to Top

Delete Cache

The delete cache link will manually delete all the cached files. The script only generates cache files if you have caching turned on.

Back to Top

Database Backup

This feature allows you to backup your database and save it to your computer.
