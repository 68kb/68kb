Categories

The categories section allows you to add, edit, and delete categories.

Adding / Editing Categories

When you add or edit categories you are shown a form with all the category fields. The only one required is the Title.

By default the uri field will be auto generated if you leave this field blank when adding or editing.

Back to Top

Deleting Categories

You can delete each category individually by clicking the delete icon.

Back to Top

Duplicating Categories

A great time saver is the ability to duplicate categories. You can click the duplicate icon to make a copy of an existing category.

Category Fields

Title (Required)
- The article title.
URI
- The category uri. This is used in the query string when viewing a category page. For example yoursite.com/category/uri. If you leave this field blank the script will auto generate the uri based off the title. You can use letters, numbers, underscores, and dashes.
Description
- The description can be used to give a brief description of the category. This is not required and if entered it will be shown on the category list page.
Parent Category:
- Select which category is the parent. The default is "No Parent".
Weight:
- By default all categories are sorted alphabetically but this can be overridden using the weight. The higher the weight the closer to the top it is.
Display:
- Can site visitors see the article.
